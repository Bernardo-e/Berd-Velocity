# ⌨️ TypeForge

> Practice faster. Type smarter.

TypeForge is a modern, beautifully designed typing practice platform that helps users improve their typing speed and accuracy through interactive exercises. Practice using random paragraphs, programming code, custom text, or even your own PDFs—all without creating an account.

## ✨ Features

* ⚡ Real-time WPM & CPM tracking
* 🎯 Live accuracy calculation
* ❌ Mistake highlighting with instant feedback
* 📄 Upload PDFs and practice directly from your documents
* 💻 Programming code typing mode
* 🧠 Multiple practice categories
* 🏆 Achievements and milestones
* 📊 Performance dashboard and history
* 📈 Progress charts
* 🔥 Daily practice streaks
* 🎨 Multiple themes (Obsidian, Cyber, Ocean, Forest, Light)
* ⌨️ Keyboard sound effects (Linear, Tactile, Typewriter)
* 💾 Local Storage support (no login required)
* 📱 Fully responsive design
* ✨ Smooth animations and modern UI
* 🕐 Custom time & word count — set any duration you want

## 🛠️ Tech Stack

* React 18
* TypeScript
* Tailwind CSS
* Framer Motion
* PDF.js (pdfjs-dist)
* Web Audio API (zero-latency key sounds)
* Local Storage

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/Bernardo-e/Berd-Velocity.git
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

## 🎯 Why TypeForge?

Unlike traditional typing websites, TypeForge focuses on making practice more personal and practical.

Practice from:

* 📄 PDFs — upload any document and type from it
* 💻 Programming code — real JS, TS, Python, Rust syntax
* 📝 Custom text — paste anything you want to practice
* 📚 Educational content — history, science, technology
* 💡 Quotes and more

Your progress stays on your device, ensuring complete privacy while delivering a fast, seamless experience.
